# Claude Code Configuration

## 作るもの

- [CTFd](https://github.com/CTFd/CTFd)の拡張機能です
- ユーザーがCTFdを使ってCTFの問題を解いた後、チームページ(/team)に行くとCertificate発行ボタンが表示される
- ユーザーがCertificate発行ボタンを押すと、ユーザーの名前とCTFの名前が記載された証明書が発行される
- 証明書はPDFで発行される
- 証明書にはユーザーの名前とチーム名と順位とスコアが記載される
- 管理者は管理画面>のプラグインページ(/admin/certificates)で証明書のデザインを設定できる
